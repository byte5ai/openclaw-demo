## Drei Befehle, dann läuft's

```bash
# Repo klonen, Dependencies installieren, Watcher starten
git clone https://github.com/byte5ai/dein-repo.git
cd dein-repo
npm install && npm run dev
```

- **`npm run dev`** öffnet den Marp-Watcher auf `localhost:8080`
- **`npm run build`** baut statisches HTML nach `dist/`
- **`npm run pdf`** erzeugt ein PDF zum Mitnehmen

> **Tipp** <span class="b5-colon">:</span> Der Code-Block hat oben rechts einen *Copy*-Button. Hover drüber, klick — schon im Clipboard.
